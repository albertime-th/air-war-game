# air-war-game
Description (描述) English: A dynamic aerial combat game built with HTML5 Canvas, featuring a universal control system that supports mouse tracking, mobile touch, and desktop keyboard. Includes real-time collision logic and automated enemy spawning.  中文： 动态 HTML5 Canvas 飞机大战游戏。具备全平台交互系统，支持鼠标追踪、手机触控及电脑键盘三种操作模式。包含实时碰撞逻辑与自动敌机生成系统。
